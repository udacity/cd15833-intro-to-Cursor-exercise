# Third-Party Images and Datasets

Udacity is committed to providing world-class, original educational content to our global community of students. To that end, ***we prefer original, author-created content.***
In some cases, however, learners are better served using third-party images and datasets created by others. We follow best practices for using and attributing third-party content, including abiding by the required usage and license agreements to add it to our content.

## Attribution Requirements

Any content that is not original, author-created content must be properly cited and attributed to its source on the classroom page, ideally with a link to the original source.
If the content is used in a video, the attribution should appear both in the video and in the classroom page below the video.

## Licensing Requirements

All third-party content must either be in the public domain or have a license that allows us to use it for commercial purposes. Acceptable licenses include:

- [CC 0](https://creativecommons.org/publicdomain/zero/1.0/) (Public Domain)
  - Attribution is still a best practice, but it is not legally required for public domain material
  - Public domain material can be remixed and used for commercial purposes
- [CC BY](https://creativecommons.org/licenses/by/4.0/deed.en)
  - BY means material must have attribution
  - Material with this license can be modified (colors changed to Udacity brand colors, dataset columns added or removed, etc.)
- [CC BY-ND](https://creativecommons.org/licenses/by-nd/4.0/deed.en)
  - BY means material must have attribution
  - ND (No Derivatives) means material with this license can **not** be modified

A Creative Commons (CC) license does not necessarily mean we are allowed to use a piece of content. Udacity content should not use any material that includes **NC** or **SA** in the license title.

- **NC** (Non-Commercial) restricts the use of the content in any activity that generates revenue. This means we cannot use it in Udacity content because Udacity is a commercial entity.
- **SA** (Share Alike) requires that any work using this content is distributed under the same license terms. This means we cannot use it in Udacity content because paywalled Udacity content is not distributed under a Creative Commons Share Alike license.

## Approved Sources of Third-Party Content

There are a few reviewed and approved sources for third-party content that may assist you in developing your course. Please note that these sources may require you to create an account, agree to their terms, and require attribution before permitting your use of their content. 

### Images

- [**Pixabay.com**](https://pixabay.com/): Check for images with the CC0 license (see details above). Other images may not be used for commercial purposes (including Udacity content). You may be able to use the images if you use the image to make a “new creative work.” See license details here: [Pixabay Terms of Service](https://pixabay.com/service/terms/)
- [**Unsplash.com**](http://Unsplash.com): All images published on *Unsplash* can be used for free. See license details here: [Unsplash License](https://unsplash.com/license)
- [**Wikipedia Public Domain images**](https://en.wikipedia.org/wiki/Wikipedia:Public_domain_image_resources): Note that not all Wikipedia images are in the public domain. Double-check the license before using an image. Most Wikipedia content will have a CC license (see details above).

### Coding Libraries & Software Packages

- Software governed by the [Apache 2.0 license](https://www.apache.org/licenses/LICENSE-2.0)
- Software governed by the [ISC License](https://opensource.org/licenses/ISC).
- Software governed by the [MIT license](https://opensource.org/licenses/MIT).

### Databases

- Databases governed by [Open Data Commons Public Domain Dedication and License (PDDL)](http://opendatacommons.org/licenses/pddl/).
- Databases governed by the Attribution License, [Open Data Commons Attribution License (ODC-By)](http://opendatacommons.org/licenses/by/).
- Databases governed by the Open Database License, [Open Data Commons Open Database License (ODbL)](http://opendatacommons.org/licenses/odbl/).
- U.S. Government Open Data available at [Data.gov](http://Data.gov).

If you have any questions about using a resource, please consult your Udacity build team.
